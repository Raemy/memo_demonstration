Changer de médecin traitant

Pour changer de médecin traitant, vous devez remplir le formulaire Cerfa n°12485 avec votre nouveau médecin lors de votre prochaine consultation. Le médecin transmet ensuite le formulaire à votre caisse.

Conséquences :

  • Sans médecin traitant déclaré, vous serez moins bien remboursé(e).
  • Le changement prend effet dès réception du formulaire par votre caisse.
  • Vous pouvez changer de médecin traitant autant de fois que nécessaire.
